# CS203A 作業說明

## 作業目標

1. **建立 GitHub 帳號**  
   - 使用專業名稱，最好是真名。

2. **建立課程專屬 GitHub 倉庫 `11401_CS203A`**  
   - 設為公開 (submission 前)  
   - 建立 `README.md`，內容包含：  
     - 課程資訊  
     - 你的姓名與學號  
     - 倉庫描述  
   - 至少有一次有意義的 commit（記錄學習內容）。

3. **建立個人網頁倉庫 `username.github.io`**  
   - 設為公開 (submission 前)  
   - 開啟 GitHub Pages，使網站可訪問  
   - 可選：連結自訂網域  

4. **使用 Markdown 文件化內容（README.md）**  
   - 使用標題、清單、連結等 Markdown 語法  
   - 清楚說明倉庫用途  

5. **完成作業提交表單**  
   - 使用元智大學信箱提交：[作業提交表單](https://forms.office.com/r/B8CUxDCyXS)
